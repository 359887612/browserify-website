require('modules') in the browser
Write some browser code with node.js-style requires: 
var foo = require('./foo');
var bar = require('../lib/bar');
var gamma = require('gamma');

var elem = document.getElementById('result');
var x = foo(100) + bar('baz');
elem.textContent = gamma(x);
Install modules with npm: 
npm install gamma
Now recursively bundle up all the required modules into a single file with the browserify command: 
browserify main.js -o bundle.js
Drop a single <script> tag into your html and you're done! 
<script src="bundle.js"></script>
  • use npm modules in the browser
  • test your modules in all the browsers with testling-ci
